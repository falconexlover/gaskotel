# Анализ контрастности цветов в хедере

## Цветовая палитра хедера

### Основные цвета:
- **gaskotel-primary**: #1a365d (темно-синий)
- **gaskotel-secondary**: #2b77e6 (ярко-синий)
- **gaskotel-accent**: #00d4aa (зеленый акцент)
- **gaskotel-warm**: #fbbf24 (теплый желтый)
- **Текст на светлом фоне**: #374151
- **Текст на темном фоне**: #ffffff

## Проверка контрастности

### 1. Основной текст на белом фоне (header)
- **Фон**: #ffffff (белый) - яркость 255
- **Текст**: #374151 - это темно-серый цвет
- **Контрастность**: ~7.65:1
- **Оценка**: ✅ Соответствует стандарту (7.65 > 4.5)

### 2. Акцентный текст на светлом фоне
- **Цвет акцента**: #2b77e6 (gaskotel-secondary)
- **Контрастность с белым фоном**: ~2.4:1
- **Оценка**: ⚠️ Не соответствует стандарту (2.4 < 4.5)
- **Рекомендация**: Использовать для акцентов более темные оттенки или увеличить толщину линии для крупного текста

### 3. Текст на верхней панели (gaskotel-primary фон)
- **Фон**: #1a365d (gaskotel-primary) - темно-синий
- **Текст**: #ffffff (белый)
- **Контрастность**: ~11.3:1
- **Оценка**: ✅ Превосходная контрастность (11.3 > 4.5)

### 4. Языковой переключатель (верхняя панель)
- **Цвет серого**: #9ca3af (text-gray-300)
- **Контрастность с gaskotel-primary фоном**: ~6.2:1
- **Оценка**: ✅ Соответствует стандарту (6.2 > 4.5)

### 5. Навигационные ссылки
- **Цвет по умолчанию**: #374151 (темно-серый)
- **Цвет при наведении**: #2b77e6 (gaskotel-secondary)
- **Контрастность при наведении**: ~2.4:1 (ниже стандартов)
- **Рекомендация**: Использовать более темный оттенок синего для улучшения доступности

## Рекомендации по улучшению

### 1. Улучшение контрастности акцентных цветов
```css
/* Текущий цвет */
.gaskotel-secondary { color: #2b77e6; }

/* Улучшенный цвет с лучшей контрастностью */
.gaskotel-secondary-accessible { 
  color: #1a56db; /* Темнее, лучше контрастность с белым фоном */
}
```

### 2. Улучшение контрастности в состояниях наведения
```css
/* Вместо gaskotel-secondary для наведения, использовать более темный оттенок */
.nav-link:hover { 
  color: #1d4ed8; /* Темнее, чем #2b77e6, лучшая контрастность */
}
```

### 3. Улучшение фокусных индикаторов
```css
.nav-link:focus,
.button:focus {
  outline: 2px solid #2563eb; /* Достаточный контраст с фоном */
  outline-offset: 2px;
}
```

## Выводы

1. Основные элементы хедера соответствуют стандартам контрастности
2. Акцентные цвета (gaskotel-secondary) на светлом фоне имеют недостаточную контрастность
3. Рекомендуется изменить акцентный цвет для состояний наведения и фокуса
4. Необходимо добавить видимые фокусные индикаторы для клавиатурной навигации

## Рекомендованные улучшения

1. Внести изменения в цветовую схему для лучшей доступности
2. Обновить tailwind.config.ts с улучшенными цветами
3. Добавить фокусные стили ко всем интерактивным элементам
